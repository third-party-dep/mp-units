Prefixes

struct prefix_family

The base for all prefix families.

Every prefix family should inherit from this type to satisfy PrefixFamily concept.

Subclassed by units::isq::iec80000::binary_prefix, units::isq::si::prefix, units::no_prefix

struct no_prefix : public units::prefix_family

No prefix possible for the unit.

This is a special prefix type tag specifying that the unit can not be scaled with any kind of the prefix.

template<typename Child, PrefixFamily PF, basic_symbol_text Symbol, ratio R>
struct prefix : public units::downcast_dispatch<Child, detail::prefix_base<PF, R>, downcast_mode::on>

A prefix used to scale units.

Data from a prefix class is used in two cases:

  • when defining a prefixed_unit its ratio is used to scale the reference unit and its symbol is used to prepend to the symbol of referenced unit

  • when printing the symbol of a scaled unit that was not predefined by the user but its factor matches ratio of a prefix from the specified prefix family, its symbol will be prepended to the symbol of the unit

Template Parameters

  • Child – inherited class type used by the downcasting facility (CRTP Idiom)

  • PF – a type of prefix family

  • Symbol – a text representation of the prefix

  • R – factor to be used to scale a unit

Subclassed by units::isq::iec80000::kibi
